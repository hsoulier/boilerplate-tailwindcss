# Rollup + Tailwind

## Initial
Clone this repository 
Install dependencies with yarn or npm 
```bash
# With Yarn 
yarn install
# Or with NPM
npm install
```

## Scripts
To watch css and js files `yarn build:watch`

To build css and js files `yarn build`

To build in production mode css and js files (minified) `yarn build:prod`
